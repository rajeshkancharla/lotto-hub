Data Source

Results are fetched directly from The Lott's official API (data.api.thelott.com). Data is cached locally in your browser for offline access.

Prediction Method

Predictions use a weighted ensemble of four statistical signals: frequency bias (hot numbers), gap analysis (overdue numbers), co-occurrence patterns (paired numbers), and position-weighted frequency. No machine learning — purely statistical.

Pattern Analysis

The app tracks frequency per ball, draws since last appearance, most common pairs, odd/even balance, and low/high split across all historical draws.
